# Studio Bjør — Website

Miami-based interior design, lighting design & custom furniture studio.

## Deploying to Vercel (free, ~10 minutes)

### Step 1 — GitHub
1. Go to [github.com](https://github.com) and create a free account if you don't have one
2. Click **New repository** → name it `studio-bjor` → click **Create repository**
3. Upload `index.html` by clicking **Add file → Upload files**
4. Click **Commit changes**

### Step 2 — Vercel
1. Go to [vercel.com](https://vercel.com) and sign in with your GitHub account
2. Click **Add New → Project**
3. Select your `studio-bjor` repository
4. Click **Deploy** — it goes live in ~60 seconds

### Step 3 — Custom Domain
1. Buy `studioBjor.com` at [Namecheap](https://namecheap.com) (~$12/yr)
2. In Vercel: go to your project → **Settings → Domains** → add `studioBjor.com`
3. Follow Vercel's DNS instructions (copy 2 values into Namecheap) — takes ~10 min to propagate
